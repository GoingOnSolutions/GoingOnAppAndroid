Overview

Zi is an App landing page created to make your App looks outstanding between the crowd. It has clear and simple design for best visual apperance and elegant feel. The features and sections let you present the App beatifuly and with its interactions, visitor will enjoy digging what your App is about and finally use it. The theme also has ultra responsive design that make the layout looks good in any screen size, thus impacting great content accesibility.

Features

  • Delivered with 5 different styles: fullscreen slider, fullscreen image (iPhone), fullscreen video, single column fullscreen image and fullscreen image (Nexus 5)
  • Simple, clean and elegant design
  • Ultra responsive design
  • Handcrafted to achieve the best readibility and accessibility
  • Working mailchimp AJAX newsletter form
  • Infinite screenshots slider
  • 7 built-in color skins
  • Vimeo and YouTube video embed ready
  • Pages with preloader and without preloader version
  • 600+ font icons
  • Easy to customize
  • HTML5 validated and CSS3 transition
  • Elegant-feel page scrolling
  • Bootstrap 3.3 compatible
  • Built with Sass and well organized HTML, CSS and Javascript files
  • PSDs provided for elements like app screenshots
  • Free support and updates
  • BONUS: Pre-launch Page with working AJAX newsletter form.

Important Notes

Note: Newsletter only works in web server with PHP environment. We recommend you to develop the theme in web server, i.e. XAMPP, MAMP or others.
Info: It's recommended to put your custom CSS into separate file to avoid changing main.css.

Fullscreen Slider

To change slide image background, go directly to the HTML and find hero unit section with class fullscreen-slider. You can find slide items inside div element. Each slide item has image as the background of the slide.

Attention: Image background will always be placed as first <img>element on the HTML structure of each slide.

Slider plugin used by this theme is Maximage 2, please visit its documentation for further at http://blog.aaronvanderzwan.com/2012/07/maximage-2-0/#options.

Colors

This theme provides built-in colors skin that you can use as starter to customize the colors. They can be found at assets/css/skins/. Don't forget to include the skin files at HTML header (assumed you have removed demo panel)

This section explains how to setting newsletter form at the page footer and pre-launch.html page as below:
